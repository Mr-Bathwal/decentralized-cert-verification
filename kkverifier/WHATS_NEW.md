## 🚀 How to Use New Features

### **CSV Batch Upload**

1. Go to `/issue` page
2. Select "Batch Issue" tab
3. Click "📄 CSV Upload" button
4. Upload your CSV file (format below)
5. Fill in certificate details
6. Click "Issue Batch Certificates"

**CSV Format**:

```csv
address
0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb
0x5aAeb6053F3E94C9b9A09f33669435E7Ef1BeAed
```

### **Understanding Verification**

1. Go to `/verify` page
2. Enter certificate ID or scan QR
3. See enhanced verification status
4. Read "How Blockchain Verification Works"
5. Understand "Why Can't Be Forged"
